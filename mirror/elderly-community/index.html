📚 学术级评估方法说明

📐 邻近性 Proximity

累积机会法，使用高斯距离衰减函数

f(d) = exp(-d²/2r²)，r随参数动态变化

🎯 多样性 Diversity

Shannon-Wiener 指数，衡量设施类别丰富度

H = -Σ(pₖ × ln(pₖ))

⚖️ 空间公平性 Gini

基尼系数衡量设施分布均衡程度

G = (ΣΣ|xᵢ-xⱼ|) / (2n²μ)

📊 集聚程度 PI

邻近指数衡量同类设施空间集聚效应

PI = ΣΣ(POIᵢ×POIⱼ/dᵢⱼ) / (ΣPOI)²

综合评分公式：

综合得分 = 0.35×邻近性 + 0.30×多样性 + 0.20×公平性 + 0.15×集聚度
